QUICK REFERENCE: CREATING IN XCODE
═══════════════════════════════════════════════════════════════

1. File > New > File > Core Data > Data Model
2. Name: PaintMyTown.xcdatamodeld
3. For each entity:
   a. Click "+" to add entity
   b. Set name, class, codegen (Manual/None)
   c. Add attributes (click "+" in Attributes section)
   d. Add relationships (click "+" in Relationships section)
   e. Set delete rules and inverses

4. Save and build (Cmd+B)
5. Run tests (Cmd+U)
